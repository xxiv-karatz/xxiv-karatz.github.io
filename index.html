Innerk Hub

University Research Collaboration Platform

A full-stack web application built to streamline research collaboration in academic environments. Innerk Hub connects researchers, facilitates project and funding management, and provides real-time communication, supervision, and progress tracking — all in one platform. Developed for COMS3003A Software Design at the University of the Witwatersrand, 2025.

Objectives

  • Centralise research collaboration in a single platform
  • Enable transparent project and milestone management
  • Facilitate supervision and reviewer feedback mechanisms
  • Track and manage research funding and grant usage
  • Empower admins with usage monitoring and role management

Key Features

  • User Verification: Firebase Google Sign-In with three roles — Researcher, Reviewer, and Admin

  • Collaboration Tools: Real-time messaging, document sharing, and milestone tracking
  • Funding Tracker: Manage grants, spending logs, and funding forecast dashboards
  • Reports & Dashboards: Export project data as CSV/PDF with custom analytics views
  • AI Recommendations (Planned): Match reviewers to projects based on expertise tags

Tech Stack

  • Frontend: React (Create React App)
  • Backend & Services: Firebase Auth, Firestore, Firebase Storage
  • Styling: Bootstrap 5 + Custom CSS
  • Testing: Jest, Codecov
  • CI/CD: GitHub Actions
  • Hosting: Microsoft Azure
  • Agile Tooling: Trello for sprint and task tracking

Testing & Quality

  • Unit testing with Jest
  • User Acceptance Testing (UAT) using Given-When-Then
  • Code coverage tracked with Codecov
  • Automated testing & deployment via GitHub Actions CI/CD pipeline

JobGenie 🧞

AI Career Assistant for South African Youth

JobGenie gives young South Africans (ages 18–30) immediate access to career guidance — closing the gap left by a shortage of career counsellors in townships and rural areas. It handles CV building, learnership discovery, interview prep, and career path recommendations through a conversational AI interface.

Key Features

  • CV Builder: Step-by-step conversation that generates a professional CV with live preview and PDF download
  • CV Checker: Upload or paste a CV, get an AI score and concrete rewrite suggestions
  • Job & Learnership Finder: SETA-aligned listings with stipend ranges and personalised apply tips by province
  • Interview Coach: Practice and mock interview modes with feedback on Confidence, Clarity, and Relevance
  • Career Quiz: 10-question quiz recommending three career paths with SA-specific next steps (SETAs, NSFAS, YES Programme)
  • Document Analysis: Upload job ads or contracts and ask JobGenie to analyse and respond

Tech Stack

  • Frontend: React 19, TanStack Start v1, Tailwind CSS v4, shadcn/ui
  • Backend: Cloudflare Workers (edge functions), Lovable Cloud
  • AI Model: Google Gemini 2.5 Flash via Lovable AI Gateway
  • Prompt approach: South-Africa-aware system prompt structured with role, context, constraints, and output format per IBM prompt engineering principles
  • PDF generation: html2pdf.js (client-side)

Sentic 📊

AI Sentiment Analysis Dashboard

Sentic analyses short text — customer reviews, support tickets, survey responses, social posts — and returns per-entry sentiment, recurring themes, executive insights, and concrete recommendations. Designed for analysts and product teams who need fast qualitative-to-quantitative conversion without a full ML pipeline.

Key Outputs

  • Per-entry sentiment label, score (–1 to 1), and confidence
  • Emotion tagging and key-phrase extraction per entry
  • Overall metrics: dominant sentiment, average score, executive summary
  • Theme extraction with mention counts and theme-level sentiment
  • 6 actionable insights and 5 recommendations
  • 6 built-in sample datasets across common domains

Tech Stack & Architecture

  • Frontend: Vanilla HTML, CSS, JavaScript — zero build step
  • Backend: Python Flask, hosted on Render
  • AI Model: DeepSeek deepseek-chat in JSON response mode (temperature 0.2)
  • Prompt approach: Strict JSON schema enforced in system prompt — per-entry sentiment, score, confidence, emotions, and key phrases; overall metrics in a single pass
  • Limits: 50 entries per request, 5,000 characters per entry

clickme.exe 🔬

AI-Powered Social Engineering Forensics

clickme.exe is a defensive security tool that deconstructs phishing messages to expose the psychological mechanics behind social engineering attacks. Paste any suspicious email, SMS, or DM and receive a structured forensic report in seconds — built for security professionals, IT/SOC teams, and awareness educators, not for generating attacks.

Key Features

  • Single Message Analysis: Paste any suspicious message for instant forensic breakdown
  • Psychological Trigger Map: Identifies urgency, fear, authority, scarcity, social proof, and 5 more manipulation tactics
  • Exploitability Score: 0–100 numeric severity rating based on likelihood of success
  • MITRE ATT&CK Mapping: Relevant technique IDs (T1566, T1566.001, etc.)
  • Technical Indicators: URLs, domains, spoofed senders, attachment warnings
  • Batch CSV Upload: Analyze up to 20 messages at once (requires a message column, max 5 MB)
  • Session Dashboard: Real-time analytics with Chart.js visualizations
  • PDF Export: Generate professional forensic reports client-side via jsPDF
  • Example Library: 7 pre-loaded real-world phishing samples (CEO fraud, invoice scams, credential phishing, crypto scams)
  • Dark/Light Theme: Toggle between modes for comfortable viewing
  • Zero Data Persistence: All session data is ephemeral; nothing is written to disk or a database

What's in a Report?

  • Psychological trigger map with detailed explanations
  • Exploitability score (0–100)
  • MITRE ATT&CK technique IDs
  • Technical indicators (URLs, domains, spoofed senders, attachments)
  • Narrative summary explaining the attack psychology
  • Defense recommendations for end-users and IT/SOC teams
  • Confidence score and risk level (low/medium/high/critical)
  • Attack category classification (CEO fraud, credential phishing, invoice scam, etc.)

Tech Stack

  • Backend: Flask (Python)
  • AI Engine: DeepSeek API (deepseek-chat)
  • Frontend: HTML5, CSS3, Vanilla JavaScript
  • Visualizations: Chart.js
  • PDF Generation: jsPDF
  • Deployment: Render (free tier)
  • Session Management: In-memory with 1-hour TTL, zero data persistence

Ethical Use

  • Defensive tool only — analyzes messages you have received, does not generate phishing content
  • Zero data persistence — all session data is ephemeral; nothing is written to disk or a database
  • Intended users: security professionals, IT/SOC teams, security awareness educators, individuals investigating messages they've received

DocuCode AI 🤖

AI-Powered Code Documentation & Testing

DocuCode AI turns messy, undocumented source code into clean, structured documentation in seconds. Paste a function or query, pick a language (or let it auto-detect), and the AI returns inline comments, a docstring, a plain-English explanation, a unit test, and a README snippet — with syntax highlighting and one-click copy.

Features

  • Inline Comments: Original code annotated line by line
  • Docstring: Google / JSDoc / Javadoc / SQL block style depending on language
  • Plain Explanation: 2–4 sentences in plain English
  • Unit Test: Runnable pytest, Jest, or JUnit snippet
  • README Snippet: Markdown section ready to paste into a repo
  • Complexity Score: simple / moderate / complex tag
  • Edge Cases: 3–6 failure modes identified by the model
  • Before vs After: Readability and maintainability scoring via /api/compare

Tech Stack

  • Frontend: React 19, TypeScript, TanStack Start v1, Vite 7
  • Styling: Tailwind CSS v4, shadcn/ui, Midnight Indigo theme
  • AI Model: Gemini 2.5 Flash via Lovable AI Gateway
  • Prompt approach: JSON-mode output with strict schema (commented_code, docstring, explanation, test_snippet, readme_snippet, complexity_score, edge_cases)
  • Runtime: Cloudflare Workers (edge)
  • Languages: Python, JavaScript, TypeScript, SQL, Go, Rust, Java — with auto-detection
