## 🧩 Project Details

### clickme.exe 🔬
*AI-Powered Social Engineering Forensics*

Defensive security tool that deconstructs phishing messages to expose psychological manipulation — urgency, fear, authority, scarcity, social proof.

**Key Features:**
- Single message analysis with instant forensic breakdown
- Psychological trigger map (10+ manipulation tactics)
- Exploitability score (0–100)
- MITRE ATT&CK mapping (T1566, T1566.001, etc.)
- Technical indicators (URLs, domains, spoofed senders)
- Batch CSV upload (up to 20 messages, 5MB limit)
- Session dashboard with Chart.js analytics
- PDF report export (client-side via jsPDF)
- 7 pre-loaded real-world phishing examples
- Dark/light theme toggle
- Zero data persistence — all session data ephemeral

**Live:** [clickme-exe.onrender.com](https://clickme-exe.onrender.com)  
**GitHub:** [xxiv-karatz/clickme-exe](https://github.com/xxiv-karatz/clickme-exe)

### JobGenie 🧞
*AI Career Assistant for South African Youth*

Built to close the career guidance gap in townships and rural areas. Serves users ages 18–30 with practical job-seeking tools.

**Features:**
- CV Builder with live preview and PDF download
- CV Checker with AI scoring and rewrite suggestions
- Job & Learnership Finder (SETA-aligned listings)
- Interview Coach with Confidence/Clarity/Relevance feedback
- Career Quiz (10 questions → 3 career paths)
- Document analysis (job ads, contracts)

**Live:** [jobgenieai.lovable.app](https://jobgenieai.lovable.app)

### Sentic 📊
*AI Sentiment Analysis Dashboard*

Fast qualitative-to-quantitative conversion for customer feedback, reviews, and social posts.

**Outputs:**
- Per-entry sentiment label, score (–1 to 1), confidence
- Emotion tagging and key-phrase extraction
- Overall metrics (dominant sentiment, average score)
- Theme extraction with mention counts
- 6 actionable insights + 5 recommendations
- 6 built-in sample datasets

**Live:** [sentic.onrender.com](https://sentic.onrender.com)  
**GitHub:** [xxiv-karatz/sentic](https://github.com/xxiv-karatz/sentic)

### DocuCode AI 🤖
*AI-Powered Code Documentation & Testing*

Paste messy code, get back production-ready documentation in seconds.

**Features:**
- Inline comments (line-by-line annotation)
- Docstrings (Google/JSDoc/Javadoc/SQL style)
- Plain-English explanation (2–4 sentences)
- Runnable unit tests (pytest/Jest/JUnit)
- README snippet (Markdown)
- Complexity score (simple/moderate/complex)
- Edge cases identified (3–6 failure modes)
- 7+ languages with auto-detection

**Live:** [docucodeai.lovable.app](https://docucodeai.lovable.app)  
**GitHub:** [xxiv-karatz/docucodeai](https://github.com/xxiv-karatz/docucodeai)
