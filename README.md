# Khulekani Mtshali – Portfolio

A modern, responsive portfolio website showcasing my work as a software engineer focused on building impactful digital experiences and supporting South African youth in their career journeys.

## 🌟 Features

- **Responsive Design**: Optimized for all devices with clean, semantic HTML and modern CSS.
- **Featured Projects**: Highlights key projects including JobGenie, Innerk Hub, and CareGuide.
- **Scrollable Project Section**: Horizontal project cards with arrow navigation for easier browsing.
- **Download Resume**: Direct download link for my resume.
- **Contact Information**: Easy access to email, GitHub, and LinkedIn.

## 🚀 Live Site

Visit the live portfolio at: [https://xxiv-karatz.github.io/](https://xxiv-karatz.github.io/)

## 📋 Featured Projects

## Sentic
Sentiment analyzer too. Analyzes customer feedback, reviews, and social posts using the DeepSeek API. Features per-entry sentiment scoring, theme extraction, insights report, and six built-in sample datasets.

- **Live App**: [sentic.onrender.com](https://sentic.onrender.com)
- **GitHub**: [xxiv-karatz/sentic] (https://github.com/xxiv-karatz/sentic)

### JobGenie 🧞
AI-powered career assistant for young South African job seekers (ages 18–30). Built in alignment with IBM prompt engineering principles.

- **Live App**: [jobgenieai.lovable.app](https://jobgenieai.lovable.app)
- **GitHub**: [5Bit-Technologies/JobGenieAI](https://github.com/5Bit-Technologies/JobGenieAI)

### Innerk Hub
Full-stack web application for university research collaboration, developed for COMS3003A Software Design at the University of the Witwatersrand.

- **GitHub**: [iamprist/University-Research-Collaboration-Platform](https://github.com/iamprist/University-Research-Collaboration-Platform)

### CareGuide 🩺
Healthcare chatbot built with Voiceflow, offering symptom guidance, mental health support, and nearby facility search with support for all 11 South African official languages.

- **Live Chatbot**: [Voiceflow development link](https://creator.voiceflow.com/share/69eb12a06698cc0dab2d182e/development)
- **GitHub**: [5Bit-Technologies/CareGuide](https://github.com/5Bit-Technologies/CareGuide)
